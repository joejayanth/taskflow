# **App Name**: TaskFlow

## Core Features:

- Task Creation: Create new tasks with title, description, priority, due date, and status.
- Rich Text Editing: Support rich text formatting for task descriptions and outcomes.
- Status Tracking: Update task status through swimlanes, transitioning between 'Yet to Start', 'WIP', 'In Review', and 'Done'.
- Priority Assignment: Assign a priority level (P0, P1, P2, P3) to each task.
- Due Date Setting: Set and track due dates for tasks using a date picker.
- Visual Prioritization: A tool that suggests which of the current tasks the user should be focusing on.
- Swim Lane Status: Display tasks in swim lanes corresponding to their status, allowing users to drag and drop cards between lanes to update the task status. Sort cards by due date (past to future).
- Card Display and Modal Details: Cards display title, due date, and status. Clicking a card opens a modal with full details.
- Status History: For each card have a 'history' section that captures the timestamp when a status was changed.

## Style Guidelines:

- Primary color: A muted sky blue (#87CEEB) to create a sense of calm and focus.
- Background color: Very light gray (#F0F0F0), almost white, to keep the focus on the content.
- Accent color: Soft lavender (#D8BFD8), for interactive elements and subtle highlights.
- Body and headline font: 'Inter', a sans-serif font known for its modern and neutral look suitable for both headlines and body text.
- Use simple, clear icons to represent task statuses and priorities.
- Card-based layout for tasks within swimlanes, with clear visual separation between elements.
- Subtle transitions and animations for status changes and task updates.
